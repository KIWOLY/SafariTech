


# SafariTech 

SafariTech is a modern ICT-focused blog platform built to help developers, students, and technology enthusiasts share and understand **emerging technologies**, **ICT concepts**, and **real-world tech issues**.

The platform aims to create a learning-driven community where ICT professionals can publish articles, explain complex topics in simple ways, and stay updated with what is happening in the tech world — especially from an **African and global perspective**.



##  Vision

To become a trusted technology knowledge hub that empowers ICT professionals and learners in Tanzania, Africa, and beyond by sharing accurate, practical, and easy-to-understand tech content.



## Features (Planned & Core)

- Publish and manage tech blog posts
-  Secure authentication using JWT
-  User roles (Admin, Author, Reader)
-  Comments and discussions on articles
-  Light & Dark mode support
-  Responsive UI for all devices
-  Future support for analytics & trending topics



##  Tech Stack

### Backend
- **Django** – Web framework
- **Django Rest Framework (DRF)** – API development
- **JWT Authentication** – Secure user authentication
- **PostgreSQL** – Relational database

### Frontend
- **React** – UI development
- **Tailwind CSS** – Utility-first styling
- **shadcn/ui** – Accessible and reusable UI components



## Project Architecture

SafariTech follows a **decoupled architecture**:

- **Backend** provides RESTful APIs
- **Frontend** consumes APIs using React
- Authentication handled via JWT
- Clean separation for scalability and maintainability



## Authentication

SafariTech uses **JWT (JSON Web Tokens)** for authentication:

* Access tokens for API requests
* Refresh tokens for session management
* Secure protected routes on both backend and frontend



## UI & Design System

* Styled using **Tailwind CSS**
* Consistent design powered by **CSS variables**
* Components built with **shadcn/ui**
* Accessible and responsive by default
* Light and Dark mode supported
